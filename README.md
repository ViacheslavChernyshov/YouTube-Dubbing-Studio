# YouTube Dubbing Studio

Локальное desktop-приложение на `PySide6` для дубляжа YouTube-видео:

- скачивает ролик по ссылке;
- извлекает и распознаёт речь;
- переводит сегменты;
- синтезирует новую озвучку;
- выравнивает её по таймкодам;
- собирает финальное видео обратно.

## Что уже есть в проекте

- GUI для запуска пайплайна и просмотра логов
- несколько TTS-движков: `Kokoro`, `Edge-TTS`, `F5-TTS`
- встроенная документация в интерфейсе
- тесты на ключевые этапы пайплайна

## Быстрый старт

### 1. Установить зависимости

```powershell
setup_env.bat
```

Или вручную:

```powershell
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

### 2. Установить FFmpeg

`ffmpeg` должен быть доступен в `PATH`.

### 3. Запустить приложение

```powershell
python main.py
```

или:

```powershell
run.cmd
```

## Важные замечания

- `models/` не коммитится: Kokoro может скачать свои ONNX-ассеты автоматически при первом запуске.
- `models/voice_presets/` тоже не обязателен для первого коммита: F5-пресеты могут быть сгенерированы на месте.
- `settings.json`, `cookies.txt`, `jobs/`, `logs/` и `.venv/` являются локальными/runtime-данными и в первый коммит не входят.
- Если нужен авторизованный доступ к YouTube, рядом с `main.py` можно положить локальный `cookies.txt`.

### Поддержка YouTube Shorts

Приложение оптимизировано для работы с YouTube Shorts:

- **Расширенные fallback клиенты**: Помимо стандартных `tv_downgraded` и `web`, теперь также используются `mediaconnect`, `mweb`, `android` и `ios` клиенты
- **Поддержка WebM форматов**: Shorts часто доступны только в WebM, поэтому добавлена полная поддержка webm/video и weba/audio комбинаций
- **Unrestricted download режим**: Если все стандартные методы не работают, приложение пробует скачать без ограничений формата как последний шанс
- **Многоступенчатая система fallback**: Форматы пробуются в порядке: MP4 → WebM → любые доступные
- **Автоматическое отключение cookies**: Если cookies.txt вызывает проблемы, приложение автоматически пробует скачать без авторизации
- **Обход блокировок**: Используются дополнительные опции yt-dlp (`--geo-bypass`, `--no-check-certificates`, специальные extractor-args) для обхода ограничений YouTube
